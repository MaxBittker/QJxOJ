The Terminal

very Powerful,
kind of terrible

learning to use a computer via the terminal comes down memorizing lots of random (and some interesting) bullshit

today i'm dipping your toe into that so that if you do encounter a CLI*, you won't be intimidated

*command line interface

launch the terminal:

Applications > Utilities > Terminal


change the color...

Shell > New Window > {Theme}

  1. You type a command
  2. Press enter/return to give that command to the computer
  3. Computer runs the program you 'invoked', and prints the results of that command onto the screen

try:

cal
some programs take inputs when you invoke them: 
banner "QJ"
say "i'm a robot"
flags are extra options that some programs take: 
say -v "Albert" "your text here"
"Agnes" "Kathy" "Princess" "Vicki" "Victoria" "Alex" "Bruce" "Fred" "Junior" "Ralph"
"Albert" "Bad News" "Bahh" "Bells" "Boing" "Bubbles" "Cellos" "Deranged" "Good News" "Hysterical" "Pipe Organ" "Trinoids" "Whisper" "Zarvox"

this example uses the -v flag (voice), the text to be spoken, and the -o flag to specify output file 
say -v "Whisper" "going into a file" -o recording.aiff

navigation

ls: "List" prints a list of folders or 'directories' 
ls
 Desktop Downloads Movies Phone Public
 Documents Library Music Pictures Sites

cd: "Change Directory" moves your terminal to the folder that you give it (everything is case sensitive) 
cd Downloads

if you don't give cd a folder name, it goes goes back to the starting folder

interacting with files

cat: "concatenate..." give it a file name from your current folder, and it will print that file's contents to the terminal 
cat examplefile.txt
 This is the text that was in that file!
 if the file was non-text format, it will 
 include a lot of random gibberish.

open: give a file or directory to open, and it will open it with the default program for that file type 
open somefile.mp3 
open Downloads

Grand Finale:

youtube-dl

  • sudo: "Super User Do" an action that requires your password (input your password and hit enter to continue)
  • curl: download a file from a url
  • -o: output that files to a place, in this case /usr/bin/, a special folder for programs 
    • sudo curl https://yt-dl.org/downloads/2015.10.24/youtube-dl 
-o /usr/bin/youtube-dl -k
    (all one line)
  • chmod: "Change Mode" set a file to be seen as a program 
sudo chmod a+rx /usr/bin/youtube-dl
if nothing went wrong... youtube-dl is now installed!
for some reason you need to update it now... 
youtube-dl -U

give it a youtube url, video code, or playlist! 
youtube-dl https://www.youtube.com/watch?v=uxpDa-c-4Mc
youtube-dl uxpDa-c-4Mc
flags: -x for audio only
